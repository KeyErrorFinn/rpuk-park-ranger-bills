Log Text Input This Log Text Input box is where you put the information you get from the in-game computer.

You can either copy and paste the data multiple times (depending on amount of pages) or just paste it all into a text file and then paste it here.
Generate
Output For Sheet This Output For Sheet box is where the generated bill output for the Senior+ Google Sheet goes.

Once the bills have been properly generated from the Log Text Input box, you just need to press the Copy button and paste the output into the Google Sheet.
Copy
Bill List This Bill List box is where all the generated bill information appears.

Once generating the bills from the Log Text Input box, each bill recipient will show here as a tab with their name and bill amount. You can expand each tab to show every item they have taken in or out of the armoury, along with the quantity and bill change for each.

After you have generated the messages from the Sheet Input box, each tab will gain two buttons. One button will be labelled as the recipient's phone number, and the other would be labelled as "Message". You can click the phone number button to copy the recipient's phone number, and click the message button to copy the customized message for that recipient.
Use Log Text Input to generate Bills
Sheet Input This Sheet Input box is where you put information from the Senior+ Google Sheet.

After pasting the Output For Sheet box content into the Google Sheet, you then copy the same data, along with the next two columns of automatically generated data here.

So you should have the columns; Name, Bill, Days active, and Phone Number for the information you already pasted.
Generate
Sample Message This Sample Message box is where the custom message that is sent to all bill recipients is set.

You can modify the message to however you want and reset it back to its original state if needed.

Custom Attributes:
- [FIRSTNAME] -> Recipient's first name
- [AMOUNT] -> Recipient's bill
